# WinNAS
轻量级局域网文件共享与协作系统
## 📁 文件管理
 - 点击「添加共享目录」将本地文件夹添加到系统中
 - 选择左侧目录后，右侧显示目录内的文件和子文件夹
 - 点击「上传文件」选择文件上传，点击「上传文件夹」选择整个文件夹上传
 - 点击「新建文件夹」在当前目录下创建子文件夹
 - 双击文件夹进入子目录，双击图片/文本文件直接预览，双击音视频文件直接播放
 - 右键文件可弹出快捷菜单，支持预览、播放、下载、分享、移动、重命名、删除
 - 点击目录右侧的 ⋯ 菜单可编辑目录属性、分享整个目录或删除目录
 - 支持按图片/视频/文档/音频等类型筛选当前目录文件
 - 图片目录可切换画廊模式浏览，支持幻灯片播放
## 🔗 分享管理
 - 在文件管理中对文件或目录点击「分享」，可创建分享链接
 - 支持设置访问密码、过期时间和最大下载次数
 - 创建成功后链接自动复制到剪贴板，可发送给局域网内其他用户
 - 分享链接支持生成二维码，手机扫码直接访问
 - 在分享管理页面可查看所有分享，支持禁用和重新启用
 - 文件夹分享支持「下载全部」打包为 ZIP 下载
## 👥 团队协作
 - 点击「创建团队」新建协作空间，可选择已有用户作为成员
 - 团队卡片右侧 ⋯ 菜单支持编辑团队属性和删除团队（仅创建者）
 - 进入团队后切换标签页：团队文件、团队成员、团队信息
 - 团队文件支持上传、新建文件夹、预览、播放、下载和删除
 - 团队成员页面可添加新成员或移除已有成员
## 💬 在线聊天
 - 左侧联系人分为「私聊」和「群聊」两个区域
 - 私聊：选择用户发起一对一聊天
 - 群聊：选择团队进入团队群聊
 - 自己发送的消息显示在右侧（蓝色气泡），对方消息显示在左侧
## ⚙️ 系统管理（仅管理员）
 - 系统信息：查看 CPU、内存、磁盘等运行状态
 - 用户管理：查看、启用/禁用用户，修改用户角色和存储配额
 - 操作日志：查看系统操作记录
 - 公告管理：发布和删除系统公告，支持置顶
 - 系统设置：修改端口、上传限制等配置参数
## 🌐 网络共享（仅管理员）
 - WebDAV：在文件资源管理器地址栏输入 WebDAV 地址，使用 WinNAS 账号密码登录
 - SMB：右键"此电脑"→"映射网络驱动器"，输入共享地址连接
 - 支持全局开关控制 SMB 和 WebDAV 服务的启用与禁用
 - 可对每个共享目录单独设置是否允许 SMB 或 WebDAV 共享
 - SMB 共享需要以管理员身份运行 WinNAS
## 🎵 媒体播放
 - 支持 MP4、WebM 等视频格式和 MP3、WAV、FLAC 等音频格式
 - 点击「播放」按钮后，页面底部出现浮动播放器
 - 播放器支持播放列表、上一曲/下一曲、进度拖动、音量调节
 - 播放器可最小化为小窗，不影响其他操作
## 🖼️ 文件预览
 - 图片：支持 JPG、PNG、GIF、BMP、WebP、SVG 格式，全屏预览
 - Markdown：预览 .md 文件时自动渲染为格式化内容
 - 文本：支持 TXT、LOG、JSON、XML、CSV、YAML、HTML、CSS、JS、PY、JAVA、C、CS 等 26 种格式，代码风格展示
## 🌙 暗黑模式
 - 点击顶部栏右侧月亮/太阳图标可切换暗黑/亮色主题
 - 主题设置会自动保存，下次打开时恢复上次选择
 - 夜间使用暗黑模式更护眼
